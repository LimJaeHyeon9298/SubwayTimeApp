# SubwayTimeApp
지하철 도착 정보 앱

## :sparkles: 프로젝트 소개

- 지하철역을 검색하면 해당 역에 언제 지하철이 도착하는지 정보를 알수 있는 앱
 




## :people_holding_hands: 멤버구성

- 개인 프로젝트 진행

## :gear: 개발 환경 

- xcode 14.3.1

- framework : Snapkit 5.6.0 , Alamofire 5.7.1 

- API : 서울시 열린데이터 광장 open API(실시간 지하철 도착정보 API)  

## :pushpin: 주요 기능 

지하철 역 검색하면 각 역을 지나는 지하철이 언제 도착하는지 알 수 있다.
